# ATM Simulator System (Java)

##  Overview

The ATM Simulator System is a desktop-based banking application developed using Java Swing and JDBC. This project simulates real-world ATM functionalities such as user authentication, deposit, withdrawal, balance enquiry, fast cash, mini statement, PIN change, and net banking.

The system provides a simple and user-friendly graphical interface and stores user and transaction details in a database.

---

##  Features

* User Login Authentication
* Account Registration (Signup)
* Deposit Money
* Withdraw Money
* Fast Cash
* Balance Enquiry
* Mini Statement
* PIN Change
* Transaction History
* Net Banking
* GUI-based ATM Interface

---

##  Technologies Used

* Java (Swing & AWT)
* JDBC
* MySQL Database
* NetBeans IDE

---

##  Project Modules

* Login.java – User authentication
* SignupOne.java – Account registration page 1
* SignupTwo.java – Account registration page 2
* SignupThree.java – Account registration page 3
* Deposit.java – Deposit money
* Withdraw.java – Withdraw money
* FastCash.java – Quick withdrawal
* BalanceEnquiry.java – Check balance
* MiniStatement.java – Show recent transactions
* PinChange.java – Change ATM PIN
* NetBanking.java – Net banking services
* Transactions.java – Transaction menu
* Conn.java – Database connection

---

##  Objective

To design and implement an ATM interface that replicates real-world banking operations using Java GUI and database connectivity.

---

##  How to Run

1. Open the project in NetBeans IDE
2. Configure MySQL database
3. Run Login.java file
4. Create account using Signup option
5. Login and use ATM services
